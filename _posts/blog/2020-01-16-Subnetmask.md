## Subnetmask

4개의 2진수 8자리수 총 32비트
192.168.0.1 = 1100 0000.1010 1000.0000 0000.0000 0001
네트워크 부분과 호스트부분으로 나뉨
하나의 로컬네트워크 = 라우터를 거쳐가는 여러개의 호스트들이 연결된 브로드 캐스트 영역

브로드캐스팅
송신 호스트가 전송한 데이터가 네트워크에 연결된 `모든 호스트`에 전송되는 방식
ex. 동네 이장님이 마이크로 "동네사람들 반상회가 있으니 모이세요"라고방송하는것
    동네 = 브로드캐스트 도메인

Prefix - 프리픽스표기란 서브넷 마스크 맨앞의 비트부터 1의개수를 표기하는 방식, 서브넷 마스크가 255.255.255.0인경우 맨앞의 비트부터 1이 24개이므로 /24로 표기

### IP 주소 클래스

#### A class 

2진수로 변환했을때 맨앞이 0인경우 (00000000 ~ 01111111 즉 0~127) A클래스가 사용하는 기본 서브넷 마스크는 255.0.0.0 이다
|<span style="color:red">0</span>NNN NNNN|

#### B class
맨앞이 10일 경우
